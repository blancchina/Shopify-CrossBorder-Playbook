# 🚀 Shopify Cross-Border E-commerce Playbook

Welcome to the ultimate playbook for scaling a direct-to-consumer (DTC) cross-border brand on Shopify. This repository is dedicated to sharing real-world strategies, tools, and code snippets that help you bypass the middlemen and sell globally.

## 🌟 Real-World Case Study: BlancChina
Building a brand is hard. Building a global brand around a 1000-year-old heritage craft is even harder. 

We are currently building **[BlancChina](https://blancchina.com)**, a Shopify-powered brand bringing authentic Dehua white porcelain (*Blanc de Chine*) directly from the kilns of Fujian, China to Western consumers and collectors.

We will use this repository to open-source our journey:
- 🛠️ **Tech Stack**: Shopify themes, vital apps, and custom Liquid snippets we use.
- 📈 **Growth SEO**: How we rank for high-intent keywords like "Guan Yin Statue" and "Kung Fu Tea Set".
- 📦 **Logistics**: Shipping fragile ceramics across the globe.
